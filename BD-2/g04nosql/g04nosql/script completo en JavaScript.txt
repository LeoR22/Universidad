Cómo ejecutarlo
Opción 1: Desde la terminal con mongosh
Guarda el script como init_g04_collections.js.
Abre la terminal.
Ejecuta: mongosh < init_g04_collections.js

Opción 2: Desde MongoDB Compass
Abre g_04 en Compass.
Ve a la pestaña Playground.
Pega el código.
Haz clic en Run.


// Selecciona la base de datos
use g_04

// 1. Crear colecciones
db.createCollection("Departments")
db.createCollection("Cities")
db.createCollection("Users")
db.createCollection("Persons")

// 2. Insertar documento de prueba en Departments

  codigo: "19",
  nombre: "Cauca",
